Pre-Defined Functions

on Synthetic Biological Components
add(element)
The add() function appends the specified element to the list of a component's elements.
Examples:
// add the Strength Property to the Promoter Part Type
Property Strength(num);
Promoter.add(Strength);
// add the Promoter Part Type to the hybridDevice hybridDevice.add(Promoter);
get(idx)
The get() function has one parameter that specifies the position of the element which should be returned. The indexing of the elements starts with the index 0.
Examples:
// get the first element of the abstractDevice, i.e., Promoter
abstractDevice.get(0);
// get the i-th element (i.e., the 4th) of the arrayDevices
num i = 3;
arrayDevices.get(i);
remove(idx)
The remove() function takes on parameter which specifies to index of the element that should be deleted from the list of elements.
Examples:
// delete the first element of the abstractDevice (i.e. the Promoter)
abstractDevice.remove(0);
// delete the second Property of the Promoter Part Type (i.e., RelativeStrength)
Promoter.remove(1);
size()
The size() function returns the size of the component. The returned value can be assigned or compared to a variable.
Examples:
// the following statement returns 4
abstractDevice.size();
// assign the size of the arrayDevices array to the nSize variable
num nSize = arrayDevices.size();
toSequence()
The toSequence() function returns a txt value that represents the DNA sequence of the biological component. If the toSequence() function is utilized on a device instance, the function returns the DNA sequence of the assembled Device. For abstract and hybrid devices, the toSequence() function inserts placeholders for the Part Types.
Example:
// print the sequence of the concreteDevice to the console
print("DNA sequence of the concreteDevice:", concreteDevice.toSequence());
// assign the sequence of the hybridDevice to the sequence txt variable
txt sequence = hybridDevice.toSequence();
